# pyzaif

This is developping now  
*pyzaif* is a python wrapper of Zaif's REST API.
[This is Document](https://corp.zaif.jp/api-docs/)

## Install
`$ pip install pyzaif`

## Usage
    import pyzaif
    api = pyzaif.API(api_key="your api_key", api_secret="your_api_secret")

    api.method1()
    api.method2(params)
    ...
